GET /api/job_invocations/:id/outputs
Get outputs of hosts in a job

Examples

POST /api/job_invocations/3/outputs
{
  "search_query": "id = 1",
  "job_invocation": {}
}
200
{
  "outputs": [
    {
      "complete": true,
      "refresh": false,
      "output": [],
      "host_id": 1
    }
  ]
}
