PUT /api/hosts/:host_id/packages/upgrade DEPRECATED
Update packages remotely using katello-agent. WARNING: Katello-agent is deprecated and will be removed in Katello 4.10. Migrate to remote execution now.

Examples

PUT /api/hosts/980190962/packages/remove
{
  "packages": [
    {
      "name": "foo"
    }
  ],
  "host_package": {
    "packages": [
      {
        "name": "foo"
      }
    ]
  }
}
400
{
  "displayMessage": "{\"name\"=>\"foo\"} is not a valid package name",
  "errors": [
    "{\"name\"=>\"foo\"} is not a valid package name"
  ]
}

Params

Param name Description
host_id
required

ID of the host

Validations:

  • Must be a number.
packages
required

list of packages names

Validations:

  • Must be an array of any type
