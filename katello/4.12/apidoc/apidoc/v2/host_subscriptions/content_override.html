PUT /api/hosts/:host_id/subscriptions/content_override
Set content overrides for the host

Examples

PUT /api/hosts/980191147/subscriptions/content_override
{
  "content_overrides": [
    {
      "content_label": "some-content",
      "value": 1
    }
  ],
  "host_subscription": {
    "content_overrides": [
      {
        "content_label": "some-content",
        "value": 1
      }
    ]
  }
}
